Anlisis de Perfil de Votantes con Teorema de Bayes

Objetivo: A partir de resultados electorales conocidos (probabilidades condicionales), el objetivo es calcular una probabilidad condicional inversa de inters estratgico.

Planteo del Problema: En el contexto de las elecciones provinciales de septiembre, se conoce la proporcin de votantes que eligi a La Libertad Avanza (LLA). Este dato es la probabilidad de apoyar a LLA, dado que se emiti un voto: $P(\text{Apoya LLA} | \text{Fue a Votar}) = 33.71\%$.

Sin embargo, para fines estratgicos, es ms til conocer una probabilidad inversa: la proporcin de simpatizantes de LLA que no participaron en la eleccin. Es decir, se busca calcular: $P(\text{No Vot} | \text{Apoya LLA})$.

1. Modelo de la Poblacin de Votantes

Para el anlisis, se construye un modelo que representa a la poblacin de individuos habilitados para votar. Este modelo se basa en los siguientes parmetros:

  • Participacin Electoral: Se utiliza el dato real de participacin del 60.98%.
  • Apoyo a LLA entre Votantes: Corresponde al resultado electoral del 33.71%.
  • Supuesto de Apoyo a LLA entre No Votantes: Se establece un supuesto plausible de que el apoyo a LLA en el grupo que no vota es del 45%, para reflejar una posible base de apoyo latente.

participacion = 0.6098
pct_lla_en_votantes = 0.3371

n_votantes = int(n_individuos * participacion)
n_no_votantes = n_individuos - n_votantes

# CORRECCIN: Se calcula el % de LLA sobre el N de votantes, no el total.
n_votantes_lla = int(n_votantes * pct_lla_en_votantes)
n_votantes_otros = n_votantes - n_votantes_lla

n_no_votantes_lla = int(n_no_votantes * pct_lla_en_no_votantes)
n_no_votantes_otros = n_no_votantes - n_no_votantes_lla

Interpretacin del rbol

El rbol visualiza el flujo de probabilidades. Ahora los valores son coherentes:

  • La probabilidad condicional $P(\text{Apoya LLA} | \text{Fue a Votar})$ es del 33.71%, tal como se defini.
  • La probabilidad conjunta (interseccin), que representa esta porcin expandida a la poblacin total, es $P(V \cap LLA) = 0.3371 \times 0.6098 = \textbf{20.56\%}$.

El problema analtico persiste: el rbol est estructurado en la direccin Voto -> Apoyo. Para calcular la probabilidad en la direccin inversa (Apoyo -> Voto), se requiere el Teorema de Bayes.

3. Aplicacin del Teorema de Bayes

El Teorema de Bayes es la herramienta formal para invertir una probabilidad condicional. La frmula es: $$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

Aplicada a nuestro problema: $$P(\text{No Vot} | \text{Apoya LLA}) = \frac{P(\text{Apoya LLA} | \text{No Vot}) \cdot P(\text{No Vot})}{P(\text{Apoya LLA})}$$

Los componentes del numerador se obtienen directamente de las ramas del rbol. El denominador, $P(\text{Apoya LLA})$, corresponde a la probabilidad total de apoyar a LLA y se calcula sumando las probabilidades de todas las rutas que terminan en "Apoya LLA".

# El denominador, P(Apoya LLA), es la suma de las intersecciones relevantes.
p_lla_total = p_voto_y_lla + p_no_voto_y_lla

print(f"Probabilidad total de apoyar a LLA, P(Apoya LLA): {p_lla_total:.2%}")
print("-" * 50)
# Aplicamos la frmula de Bayes
numerador = p_lla_dado_no_voto * p_no_voto
denominador = p_lla_total
prob_bayesiana_final = numerador / denominador

print(f"Resultado del Teorema de Bayes:")
print(f"P(No Vot | Apoya LLA) = {prob_bayesiana_final:.2%}")

Conclusin

El anlisis indica que un 46.06% de los individuos que apoyan a LLA no particip en la eleccin provincial.

Este resultado tiene una implicacin estratgica directa: sugiere que una parte significativa de la base de apoyo del partido no est movilizada. Para futuras elecciones, una campaa enfocada en aumentar la participacin de este segmento podra tener un impacto considerable en el resultado final. El Teorema de Bayes permite cuantificar la magnitud de este grupo, convirtiendo datos electorales en informacin accionable.
