Diccionarios¶

Datos Simples y Compuestos¶

Datos Simples: entero (int), los nmeros de punto flotante (float) y los valores booleanos (boolean).

Datos Compuestos: comprenden aquellos datos con elementos de valores de un mismo tipo o de diferentes tipos, que se representan unificados para ser guardados o procesados. Ejemplos de ellos son cadenas de caracteres (string), listas, tuplas, diccionarios.

Datos inmutables y mutables¶

Las secuencias de datos pueden ser inmutables o pueden ser mutables.

Una secuencia de datos inmutable es un conjunto finito de datos cuyos valores y orden de aparicin no pueden modificarse una vez asignados, como por ejemplo los string.

Diccionarios¶

Un diccionario es una lista no ordenada de trminos (clave valor), lo que denominaremos pares. Para cada uno de los trminos incluidos en el diccionario se pueden definir distintos valores asociados.

Al igual que las listas, los diccionarios son mutables, es decir podemos agregar, quitar o modificar los elementos del diccionario en tiempo de ejecucin. Los diccionarios pueden estar contenidos en listas y viceversa.

La principal diferencia con las listas es que accedemos a los elementos de los diccionarios mediante claves y no a travs de su posicin. Podemos decir que un diccionario es una matriz asociativa (tambin conocida como hash). Cualquier clave del diccionario est asociada (o mapeada) a un valor. Los valores de un diccionario pueden ser cualquier tipo de datos de Python (enteros, listas, cadenas, tuplas, otros diccionarios, etc).

Las claves son nicas dentro de un diccionario, mientras que los valores pueden no serlo. Los valores de un diccionario pueden ser de cualquier tipo, pero las claves deben ser de un tipo de datos inmutable, como cadenas, nmeros o tuplas.

Más sobre Diccionarios¶

A - No se permite ms de una entrada por clave. Esto significa que no se permiten claves duplicadas. Cuando se encuentran claves duplicadas durante la asignacin, la ltima asignacin pisa la anterior.

diccionario_err['Uno']
Out[1]:
1

B - Las claves deben ser inmutables. Esto significa que puede usar cadenas, nmeros o tuplas como claves de diccionario.

Creación de un Diccionario¶

Método 1: Explícito¶

In [55]:
votos = {"candidatoA" : 200, "candidatoB" : 150, "candidatoC" : 100, "candidatoD" : 50}
print(votos)

{'candidatoA': 200, 'candidatoB': 150, 'candidatoC': 100, 'candidatoD': 50}

Método 2: A partir de un diccionario vacío¶

votos["candidatoA"] = 200
votos["candidatoB"] = 150
votos["candidatoC"] = 100
votos["candidatoD"] = 50

Método 3: A partir de la función zip()¶

In [12]:

Diccionarios Anidados¶

Podemos almacenar dentro de un diccionario otro diccionario. En este caso, cada valor asociado a una clave es en s mismo un diccionario.

Candidatos = {
  "CandidatoA" : {
    "nombre" : "Tony",
    "apellido": "Blair",
    "anio" : 1953
  },
  "CandidatoB" : {
    "nombre" : "William",
    "apellido": "Hague",
    "anio" : 1961
  },
  "CandidatoC" : {
    "nombre" : "Charles",
    "apellido": "Kennedy",
    "anio" : 1959
  }
}

Para acceder al valor de una de las claves de un diccionario interno, se usa el operador de indexacin anidada [clave1][clave2]

Métodos de los diccionarios¶

Existen varios mtodos aplicables a diccionarios

dict.clear (): Elimina todos los elementos del diccionario dict

dict.copy (): Devuelve una copia superficial del diccionario dict

dict.fromkeys (): Cree un nuevo diccionario con claves de seq y valores establecidos en value .

dict.get (clave, predeterminado = Ninguno): Para clave clave, devuelve valor o predeterminado si la clave no est en el diccionario

dict.items (): Devuelve una lista de pares de tuplas de dict (clave, valor)

dict.keys (): Devuelve la lista de claves del diccionario dict

dict.setdefault (clave, predeterminado = Ninguno): Similar a get (), pero establecer dict [clave] = predeterminado si la clave no est ya en dict

dict.update (dict2): Agrega pares clave-valor del diccionario dict2 a dict

dict.values (): Devuelve la lista de valores de diccionario dic
